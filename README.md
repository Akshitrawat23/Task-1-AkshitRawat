# ☁️ Static Website Hosting on AWS S3

## 🚀 Project Overview

This project demonstrates the deployment of a **static portfolio website** using **Amazon Web Services (AWS) S3 Static Website Hosting**. The website was developed and hosted as part of an internship task assigned by **DecodeLabs**, providing hands-on experience with cloud deployment and storage services.

---

## ✨ Features

* 🌐 Static website hosting using AWS S3
* 🔓 Public website accessibility
* ☁️ Cloud-based deployment
* 📈 Scalable and reliable hosting infrastructure
* 👨‍💻 Personal portfolio showcasing skills and projects
* ⚡ Fast and lightweight web hosting

---

## 🛠️ Technologies Used

* 📝 HTML
* 🎨 CSS
* ☁️ AWS S3
* 🖥️ AWS Management Console

---

## 🏗️ Project Architecture

1. 📦 Created an Amazon S3 bucket
2. 📤 Uploaded website files (HTML, CSS, and assets)
3. ⚙️ Enabled Static Website Hosting
4. 🔐 Configured bucket permissions and public access policies
5. 🚀 Deployed the website and verified public accessibility
